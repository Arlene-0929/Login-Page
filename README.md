# Login Page - README

## Overview
A modern, responsive login page with a clean UI design. This project demonstrates frontend development skills with a minimalist approach using HTML and CSS only.

- **Responsive Design**: Works on all device sizes (mobile, tablet, desktop)
- **Modern UI**: Clean layout with a background image

- **Minimalist Approach**: Lightweight with no external dependencies

- **HTML5**: Semantic markup

- **No JavaScript**: Pure HTML/CSS implementation
- **Self-hosted Assets**: All images included in the repository

## Installation & Local Development
1. Clone the repository:
   ```bash
   git clone https://github.com/Arlene-0929/Login-Page.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Login-Page
   ```
3. Open `index.html` directly in your browser or use a live server

## Usage
Simply open `index.html` in any modern web browser. The login form is static (frontend only) and serves as a UI demonstration.

## Design Details
- **Layout**: Centered form with full-screen background
- **Colors**: Clean white form with dark text on gradient background
- **Typography**: System fonts stack for optimal performance
- **Spacing**: Consistent padding and margins throughout

## Deployment
This project is automatically deployed on **Vercel** through GitHub integration. Any changes pushed to the main branch will trigger a new deployment.

## Customization
To customize this login page:

1. **Change Background Image**:
   - Replace `assets/images/login-illustration.jpg` with your own image
   - Update the CSS `background-image` URL in `style.css` if needed

2. **Modify Colors**:
   - Edit CSS variables in `:root` selector in `style.css`

3. **Change Form Elements**:
   - Edit `index.html` to add/remove form fields
   - Update corresponding styles in `style.css`
